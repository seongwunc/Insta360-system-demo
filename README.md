# Insta360 一体化营销工作台 Demo

## 页面打开

- 在线页面：`https://seongwunc.github.io/Insta360-system-demo/`
- 本地预览：

```cmd
npm install
npm run dev
```

- 本地构建：

```cmd
npm run build
```

## 文档目录

- `demo/index.html`：页面入口
- `demo/app.js`：页面逻辑、跳转关系、交互状态
- `demo/styles.css`：页面样式
- `vite.config.js`：构建配置
- `package.json`：项目脚本和依赖
- `.github/workflows/deploy.yml`：GitHub Pages 自动部署
- `README.md`：项目说明

## 说明

- `dist/` 为构建产物，不需要手动维护
